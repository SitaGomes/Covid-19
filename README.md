## :computer: Instalation

```bash
// First clone the repository
$ git clone http://github.com/SitaGomes/WeatherCast

// Them go to the project folder
$ cd Mobilus



// In the back-end
$ cd backend

// Install back-end dependencies
$ npm i

// Turn the server on
npm run server



// In the front-end
$ cd frontend

// Install front-end dependencies
$ npm i

// Start the site
$ npm start

```
